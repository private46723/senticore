# **App Name**: Simple Page

## Core Features:

- Page Creation: Create a new page with basic HTML structure using Next.js.
- Content Display: Display 'Merhaba Dünya' (Hello World) on the page.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to create a calm and friendly atmosphere.
- Background color: Very light blue (#F0F8FF) to maintain a clean, unobtrusive background.
- Accent color: Pale violet (#BDB2FF) to add visual interest without being too loud or distracting.
- Body and headline font: 'PT Sans' for a balance of modernity and warmth, suitable for short text.
- Simple, centered layout for the 'Merhaba Dünya' text.
